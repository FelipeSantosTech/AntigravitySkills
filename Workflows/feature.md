---
description: Design and implement a feature.
---
1. Understand the user's requirements for the feature.
2. Propose an architecture emphasizing small SwiftUI views and testable ViewModels.
3. Write the necessary tests (if applicable) and implement the feature in small, iterative steps.
4. Ensure the UI aligns with modern iOS aesthetics (no storyboards).
5. Request a review from the user to confirm it meets expectations.
